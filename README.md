# HR_Tableau
👥 HR Analytics Dashboard – Tableau Project
An interactive HR Analytics Dashboard built in Tableau to analyze employee workforce data, attrition trends, job satisfaction, and demographic insights. This dashboard helps HR teams and business managers identify patterns in employee turnover and make data-driven workforce decisions.

📌 Project Overview
The dashboard provides a comprehensive view of employee data across departments, age groups, education fields, and job roles. It focuses on understanding:

Employee attrition patterns
Workforce demographics
Job satisfaction levels
Department-wise employee distribution
Gender-based attrition analysis

The project transforms raw HR data into meaningful visual insights using interactive Tableau visualizations and filters.

🎯 Business Insights Generated
Identified departments with the highest attrition rates
Analyzed which age groups are more likely to leave
Compared attrition trends between male and female employees
Evaluated employee satisfaction across job roles
Discovered educational backgrounds contributing to higher turnover

🚀 Purpose of the Project
The goal of this project is to demonstrate data visualization and analytical skills by converting HR datasets into meaningful business insights using Tableau. This dashboard can assist organizations in improving employee retention strategies and workforce management.
